# Paquetes Populares de pub.dev

Guía de paquetes más usados en proyectos Flutter profesionales.

## State Management

| Paquete | Uso | Complejidad |
|---------|-----|-----------|
| **provider** | State simple | ⭐⭐ |
| **riverpod** | State avanzado | ⭐⭐⭐ |
| **bloc** | Arquitectura compleja | ⭐⭐⭐⭐ |
| **getx** | Todo en uno | ⭐⭐⭐ |

```bash
flutter pub add provider
flutter pub add riverpod
flutter pub add flutter_bloc
flutter pub add get
```

## Networking

```bash
flutter pub add http           # Requests HTTP
flutter pub add dio            # Cliente HTTP avanzado
flutter pub add retrofit       # API rest tipado
```

## Base de Datos

```bash
flutter pub add sqflite        # SQLite
flutter pub add hive           # Local storage rápido
flutter pub add firebase_database  # Firebase Realtime DB
```

## Firebase

```bash
flutter pub add firebase_core
flutter pub add firebase_auth
flutter pub add cloud_firestore
flutter pub add firebase_storage
```

## UI/UX

```bash
flutter pub add google_fonts   # Fuentes Google
flutter pub add cached_network_image  # Caché de imágenes
flutter pub add lottie         # Animaciones Lottie
flutter pub add flutter_svg    # Iconos SVG
```

## Validación y Serialización

```bash
flutter pub add form_builder_validators
flutter pub add json_serializable
flutter pub add freezed_annotation
```

## Testing

```bash
flutter pub add mockito
flutter pub add integration_test
```

## Utilidades

```bash
flutter pub add intl          # Internacionalización
flutter pub add url_launcher  # Abrir URLs
flutter pub add share_plus    # Compartir contenido
flutter pub add device_info_plus  # Info del dispositivo
```

## Instalar Múltiples Paquetes

```bash
flutter pub add provider riverpod http dio sqflite firebase_core
```

## Buscar Más

- Web: https://pub.dev
- Comando: `flutter pub search [nombre]`
